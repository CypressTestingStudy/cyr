Steps to install:

npm install

Project Structure
- All in English
- All in CamelCase
***
**Variables Names:**
	Button: btn
  Click: clk
  Screen: scr
  Source: src
  List: List
  Dropdown list: ddl
	
***
**Constrains:**
	The first letter should will not go in UpperCase (class, methods, variables, etc)
	It does not use VAR type, hyphen, and underlines.
	Pointers must go in variables.
